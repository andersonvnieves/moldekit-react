# Getting Started

This guide will help you quickly set up and use MoldeKit components in your project.

## Installation

To install MoldeKit, use npm or yarn:

```bash
npm install moldeKit-react
```

```bash
yarn install moldeKit-react
```

## Prefix Convention

All MoldeKit React components use the `mk-` prefix for clarity and to avoid naming conflicts. For example:

- `mk-button`
- `mk-card`
- `mk-input`
