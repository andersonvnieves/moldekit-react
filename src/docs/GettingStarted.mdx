# Getting Started

This guide will help you quickly set up and use MoldeKit components in your project.

## Installation

To install MoldeKit, use npm or yarn:

```bash
npm install moldekit-react
```

MoldeKit depends on Lucid Icons to work, install this dependency too:

```bash
npm install lucide-react
```

## Usage

After installing moldekit-react, import the global CSS in your appâ€™s entry point (`main.tsx`or `App.tsx`):

```tsx
import 'moldekit-react/css';
```

or on your main `.css` file:

```css
@import 'moldekit-react/css';
```
Then import and use components in your `.ts` or `.tsx` files:

```tsx
import { MkButton } from 'moldekit-react';

function MyComponent() {
  return (
    <MkButton onClick={() => console.log('clicked')}>
      Click Me!
    </MkButton>
  );
}
```

> Note: CSS import goes once at app root; components import per-file as needed.


## Prefix Convention

All MoldeKit React components use the `Mk` prefix for clarity and to avoid naming conflicts. For example:

 `MkButton`
 `MkCard`
 `MkInput`
